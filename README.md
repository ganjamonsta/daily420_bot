## Как запустить

### Windows (рекомендуемый способ)

```
start.bat
```

Скрипт создаст venv, установит зависимости и запустит бота.

### Вручную

```bash
pip install python-telegram-bot==22.* sqlalchemy[asyncio] aiosqlite python-dotenv
cp .env.example .env
# отредактируй .env — впиши BOT_TOKEN от @BotFather
python bot.py
```

### Debian 12 (сервер, systemd + менеджер)

1. Клонируй репо в `/opt/daily420_bot`:

```bash
sudo mkdir -p /opt
cd /opt
sudo git clone <YOUR_REPO_URL> daily420_bot
cd /opt/daily420_bot
```

2. Запусти автоустановку:

```bash
sudo bash deploy/install_debian12.sh
```

Скрипт ставит сервис от пользователя, который вызвал `sudo` (например, `ganja`).

3. Если токен ещё не задан, отредактируй `.env`:

```bash
sudo nano /opt/daily420_bot/.env
# BOT_TOKEN=123456:ABC...
```

4. Перезапусти сервис:

```bash
sudo systemctl restart daily420-bot
sudo systemctl status daily420-bot --no-pager
```

#### Менеджер бота (CLI)

После установки доступна команда:

```bash
sudo daily420-bot help
```

Основные команды:

- `sudo daily420-bot status` — статус сервиса
- `sudo daily420-bot logs 200` — последние 200 строк логов
- `sudo daily420-bot follow` — live-логи
- `sudo daily420-bot restart` — рестарт
- `sudo daily420-bot update` — безопасный апдейт (`git pull --ff-only` + deps + restart)
- `sudo daily420-bot update-force` — форс-апдейт (reset/clean + deps + restart)
- `sudo daily420-bot menu` — интерактивное меню

Сервис использует unit-файл: `deploy/daily420-bot.service`.

## Лор‑система

### Три культуры

1. **🏔️ Гималаи** — Шива, садху, Atharva Veda, Парвати
2. **🕌 Шёлковый путь** — суфии, Аль‑Маюси, Марокко, Багдад
3. **🇺🇸 420‑Америка** — Waldos (1971), Grateful Dead, Калифорния

### Как лор встроен в механику

- **Культурные фразы** — растение «говорит» голосом текущей культуры
- **Исторические факты** — 25% шанс при каждом действии (21 факт, по 7 на культуру)
- **Лор‑бонусы** — Благословение Шивы, Суфийский ветер, 4:20 на часах
- **Лор‑проблемы** — Гималайский мороз, Песчаная буря, Рейд DEA
- **Стадийные нарративы** — развёрнутый текст при смене стадии
- **Лор‑удобрения** — Пепел из храма Шивы, Марокканский фосфат и т.д.
- **ЛОР‑магазин** — Гималайский грунт, Калифорнийские нутры
- **`/lore`** — полная история Духа в 4 главах
- **`/fact`** / кнопка «📜 Факт» — рандомный исторический факт
